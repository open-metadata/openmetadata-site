## Changelog

### Improvements

-   Improve PII auto-classification accuracy by refining pattern recognition for credit cards, driving licenses, and tax file numbers with better confidence scoring.
-   Automatically assign approval tasks to newly patched reviewers on entities like Data Contracts, Data Products, and Metrics.
-   Add Assets API for Domain, DataProduct, Tag, GlossaryTerm, User, and Team entities while deprecating inline assets field.
-   Add data retention policies for test case results and profile data with configurable cleanup periods.
-   Add tag usage cleanup functionality to data retention with relationship validation and orphaned tag removal.
-   Upgrade Axios to 1.12.0 to fix security vulnerability and update js-yaml dependency.
-   Add stream closure API and ensure log buffers are flushed when ingestion pipelines complete.
-   Add multi-repository support for Looker connector with LocalRepositoryPath credential option.
-   Switch Presidio recognizers from multilingual to English-specific spaCy model for improved PII detection performance.
-   Add Collibra Connector. (Collate)
-   Add email templates loading for support-email-invitaiton template. (Collate)

### Fixes

-   Fix email configuration templates default value from 'collate' to 'openmetadata'.
-   Fix column case sensitivity issue between dbt and Snowflake by adding support for case-insensitive column validation.
-   Fix SearchBar component race conditions by using refs to track search state and prevent stale closures in debounced callbacks.
-   Fix search filtering for domain and data product by separating classification and glossary tags fields in search indices and simplifying frontend filter logic.
-   Fix Domain and Data Product UI issues including redirect paths, status badge visibility, and domain query filtering.
-   Fix authentication validators to use correct client ID field (AUTH_CLIENT_ID) for public clients and improve Okta introspection URL handling.
-   Fix App Resource run ordering by start time and consolidate S3 Server-Side Encryption configuration logic.
-   Fix lineage pagination and rendering with direction-based count logic to optimize performance.
-   Fix: Removed ignored default value Approved, Updating Reviewer tasks, and field extractors. (Collate)
