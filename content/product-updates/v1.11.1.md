## Changelog

### Fixes

- SAP HANA Connector now correctly identifies view names by reading the resourceUri attribute instead of the data source ID
- Fixed authentication issue for entities with 'version' in name that was incorrectly skipping auth headers
- Fixed domain widget issue in User Details Page
- Fixed subdomains data visibility issue due to query params
- Fixed Glossary term circular reference issue with CSV import
- Fixed Sagemaker unified studio registered models
- Fixed dbtcloud lineage issues and added environment ID filter
- Fixed profiler processor status logic to show partial success at metric/column level
- Fixed Snowflake profiler to use connection account usage setting for system metrics
- Fixed using wrong indices for lineage queries
- Fixed OIDC session expiry to ensure minimum 1 hour
- Fixed js-yaml version to resolve security vulnerability
- Fixed restoring soft deleted assets from ingestion

### Features

- Enhanced Impact Analysis with pagination and improved filters
- Added support for multiple classifications in TagProcessor
- Added Edit Icon to Update Assignee directly from Incident Manager Page
- Added search functionality for data type selection in profiler configuration
- Added custom properties hint description tooltips for better user guidance
- Added dbtcloud environment ID filter and fixed lineage issues
- Added support for MSSQL SSL connections
- Added Collate Credits feature
- Added Flowable History Timed Cleanup and Ops Command for cleanup

### UI Improvements

- Fixed right panel across page occurrences with improved width and styling
- Addressed Explore card feedbacks with improved UX and navigation
- Enhanced right panel UX with improved styling and domain edit restriction for data products
- Added missing entity types support in EntitySummaryPanel and right panel
- Side panel now handles nested structure in right entity panel for schema tab
- Replaced static name with brandName throughout the UI for better branding
- Disabled name column in bulk edit actions to prevent accidental changes
