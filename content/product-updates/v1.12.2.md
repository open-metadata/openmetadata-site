## Changelog

### Improvements

- **Data Observability:** Improved layout responsiveness for test case and test suite summary cards in Data Quality [#26402](https://github.com/open-metadata/OpenMetadata/pull/26402)
- **Data Observability:** Migrated Data Insights entity pagination from offset-based to keyset pagination to prevent sort buffer exhaustion [#26127](https://github.com/open-metadata/OpenMetadata/pull/26127)
- **Discovery:** Users now inherit personas from their team memberships [#26052](https://github.com/open-metadata/OpenMetadata/pull/26052)
- **Integration:** Added support for filtering cold storage files during datalake ingestion [#26137](https://github.com/open-metadata/OpenMetadata/pull/26137)
- **Integration:** Improved error handling to mark ingestion agents as failed when quota limits are exceeded [#26058](https://github.com/open-metadata/OpenMetadata/pull/26058)
- **Integration:** Enhanced Kafka Connect lineage to support wildcard file name matching for storage services [#26034](https://github.com/open-metadata/OpenMetadata/pull/26034)
- **Integration:** Added StorageServices entity support to the Python client SDK [#26164](https://github.com/open-metadata/OpenMetadata/pull/26164)
- **Data Observability:** Increased notification template body size limit to 64KB [#26094](https://github.com/open-metadata/OpenMetadata/pull/26094)
- **Platform:** Improved server URL sanitization for deploy pipeline API calls [#26114](https://github.com/open-metadata/OpenMetadata/pull/26114)
- **Data Observability:** Enabled Quartz JDBC clustering to prevent duplicate event processing in high availability deployments [#26012](https://github.com/open-metadata/OpenMetadata/pull/26012)
- **Integration:** Added support for dataflow to database table lineage in Power BI [#26291](https://github.com/open-metadata/OpenMetadata/pull/26291)
- **Platform:** Improved Azure AD SSO logout to properly terminate IdP sessions [#26327](https://github.com/open-metadata/OpenMetadata/pull/26327)
- **Integration:** Limited records summary logging to prevent out-of-memory failures in ingestion agents [#26167](https://github.com/open-metadata/OpenMetadata/pull/26167)
- **Platform:** Implemented canvas-based edge rendering to significantly improve column-level lineage page performance for large tables [#25930](https://github.com/open-metadata/OpenMetadata/pull/25930)
- **Governance:** Activity feeds now correctly record the actual user for bulk asset operations on domains, data products, and teams [#26266](https://github.com/open-metadata/OpenMetadata/pull/26266)
- **Data Observability:** Pipeline logs breadcrumb and title now display entity display names instead of IDs [#26331](https://github.com/open-metadata/OpenMetadata/pull/26331)
- **Discovery:** Service search now supports searching by display name in addition to service name [#26399](https://github.com/open-metadata/OpenMetadata/pull/26399)
- **Data Observability:** Aligned observability alerts and incident manager header styles with data quality page design [#26467](https://github.com/open-metadata/OpenMetadata/pull/26467)
- **Discovery:** Search placeholders in service tabs now dynamically reflect the asset type instead of the service name [#26511](https://github.com/open-metadata/OpenMetadata/pull/26511)
