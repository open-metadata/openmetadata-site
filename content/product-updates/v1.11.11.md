## Changelog

### Improvements

- Added support for ingesting metadata from a local JSON/CSV file via the REST connector instead of only a remote URL [#26036](https://github.com/open-metadata/OpenMetadata/pull/26036)
- Improved Snowflake ingestion performance by no longer fetching stage definitions, reducing unnecessary API calls [#26009](https://github.com/open-metadata/OpenMetadata/pull/26009)
- Improved test case summary list loading and rendering performance [#26004](https://github.com/open-metadata/OpenMetadata/pull/26004)

### Fixes

- Fixed Data Product description area scrolling when content is long [#26040](https://github.com/open-metadata/OpenMetadata/pull/26040)
- Fixed tag input clearing previously added tags when pressing Enter [#26039](https://github.com/open-metadata/OpenMetadata/pull/26039)
- Fixed Global Domain Filter label and options to show correct translated text [#25982](https://github.com/open-metadata/OpenMetadata/pull/25982)
- Fixed user dropdown in header to display translated strings correctly (8baf4897fe)
- Fixed Data Product form validation to scroll to error when validation fails [#25760](https://github.com/open-metadata/OpenMetadata/pull/25760)
- Fixed PATCH requests to prepare pipeline for deploy failing due to missing service reference [#25984](https://github.com/open-metadata/OpenMetadata/pull/25984)
- Fixed test suite pipeline completion to correctly trigger ChangeEvents for related test suites [#25910](https://github.com/open-metadata/OpenMetadata/pull/25910)
- Fixed domain filters in search and discovery to return correct results from Elasticsearch [#26037](https://github.com/open-metadata/OpenMetadata/pull/26037)
- Fixed table DDL generation with special characters in names (e.g. spaces, reserved words) [#26007](https://github.com/open-metadata/OpenMetadata/pull/26007)
- Fixed ingestion pipelines failing with ValidationError when using Pydantic v2–compatible configs [#26014](https://github.com/open-metadata/OpenMetadata/pull/26014)
- Fixed dbt lineage and tests sync by using latest run_results entry when multiple entries exist for the same unique_id [#25996](https://github.com/open-metadata/OpenMetadata/pull/25996)
- Fixed Datalake connector errors so ingestion runs reliably [#25953](https://github.com/open-metadata/OpenMetadata/pull/25953)
