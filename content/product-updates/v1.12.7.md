## Changelog

### ✨ Improvements

- Platform: Enhanced search index performance and reduced memory footprint during reindex operations to minimize runtime impact
- Platform: Improved reliability of bulk indexing with better handling of large payloads and batching
- Platform: Improved reindex process memory usage and lowered thread priority to reduce application impact
- Platform: Improved async event processing reliability during high-volume metadata operations
- Platform: Enhanced metadata synchronization and indexing resiliency for large workloads
- Governance: Refactored asset certification storage model for improved data integrity and reliability
- Platform: Added SAML metadata XML upload support to simplify enterprise SSO configuration
- AI: Enforced payload size limits for vector embedding bulk operations to improve semantic search stability
- Ingestion: Improved Looker ingestion diagnostics and troubleshooting visibility
- UI: Improved breadcrumb rendering and metadata navigation for metric entities
- Platform: Improved authorization handling for operational and deployment APIs

### 🔧 Fixes

- Platform: Fixed race conditions in async consumers during rapid create/delete operations
- Platform: Fixed intermittent failures during metadata indexing and synchronization workflows
- Platform: Fixed ingestion stability issues related to large payload handling
- Platform: Fixed reliability issues during concurrent bulk indexing operations
- UI: Fixed malformed activity feed rendering and JSON parsing issues
- UI: Fixed entity navigation issues caused by invalid activity payloads
- Security: Prevented sensitive workflow configuration data from being exposed in API responses
- Security: Enforced bot-type validation on token generation endpoints to prevent unauthorized JWT creation
- Security: Upgraded MCP Java SDK dependency to address known vulnerabilities
- Security: Upgraded Netty dependencies to address multiple CVEs
- Security: Upgraded frontend dependencies including Vite to address known vulnerabilities
- Authentication: Fixed MSAL token renewal issues on Safari causing unexpected session loss
- Authentication: Ensured auth tokens are refreshed before retrying failed requests
- Platform: Fixed MCP OAuth callback servlet registration fallback handling
